# FRC Team 449 Website (robot.mbhs.edu)

Grav CMS site for FRC Team 449, The Blair Robot Project. The live site runs
from flat files on its own server; this repo holds the site's documentation
and a backup of page content. See docs/RUNBOOK.md for how the site actually
runs and is maintained.

- **Live site:** https://robot.mbhs.edu
- **Documentation:** see [`docs/`](docs/) — start with `docs/CLAUDE.md`
- **Theme repo:** https://github.com/blair-robot-project/grav-theme-mod-quark
- **Grav CMS docs:** https://learn.getgrav.org
