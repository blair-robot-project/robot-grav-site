Frozen Grav installation skeleton
Last updated: 2026-07-03

This is a point-in-time copy of the generic Grav CMS installation
(core code, config, plugins, themes, accounts) as it existed when this
repo was first set up. It has not been kept in sync with the live
site and should not be treated as current.

The live site at robot.mbhs.edu runs its own independent copy on its
server. This folder is kept for historical reference only. See
docs/RUNBOOK.md for how the live site actually works.
